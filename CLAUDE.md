# CLAUDE.md

## 项目简介

这是一个 mini Claude Code 的 CLI 工具，用 TypeScript/Node.js 实现，支持 OpenAI 兼容接口。

项目的核心设计目标是**高可扩展性**和**高解耦性**。模块之间通过定义好的接口通信，每个模块可以独立开发和测试。

完整的架构设计、目录结构、模块依赖关系和实现计划详见 `plan.md`。

---

## 代码风格

- **易读优先**：在不影响功能和性能的前提下，代码应该让人容易理解。避免过度简洁或炫技式写法。
- **异步处理**：使用 `async/await`，避免 `.then()` 链。
- **注释**：复杂逻辑必须写注释说明意图，函数/类导出建议写简短的说明注释。

---

## 接口约定

模块之间通过接口解耦。核心原则：

- **改模块内部实现时**，不要改动模块对外暴露的接口
- **如果必须改接口**，需要意识到这会影响其他依赖该接口的模块，并同步更新

每次开始写一个新模块之前，先讨论并确认该模块对外暴露的接口，写完后补充到下方的"模块接口速查"中。
